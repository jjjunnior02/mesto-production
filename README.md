# Mesto
Проектная работа курса Яндекс.Практикума. Интерактивная страница, где пользователь может редактировать профиль, добавлять/удалять карточки мест, ставить лайки и менять аватар. Все данные хранятся на сервере Яндекс.Практикума через REST API.

## Ссылки
Репозиторий проекта: https://github.com/jjjunnior02/mesto-production
Опубликованный проект: https://jjjunnior02.github.io/mesto-production/

## Запуск проекта локально
npm install
npm run dev

## Сборка проекта
npm run build
Готовые файлы появятся в папке `dist`.

## Публикация на GitHub Pages:
npm run deploy
